# Timestamp - 时间工具

一个实用的时间工具，支持时间戳转换、时区转换、时间计算等多种功能。

## 功能特点

- ⏰ **当前时间显示**：实时显示本地时间、时间戳和时区
- 🔄 **时间戳转换**：日期时间与时间戳之间的双向转换
- 🌍 **多时区显示**：同时显示多个常用时区的当前时间
- 🔀 **时区转换**：输入一个时间，查看它在多个时区的对应时间
- 📊 **时间计算**：计算两个时间点之间的差值（天数、小时、分钟等）
- 📅 **工作日计算**：计算两个日期之间的工作日数量
- ⏱️ **相对时间**：显示"3天前"、"2小时后"等相对时间
- 🚀 **快捷操作**：常用时间快捷按钮（现在、今天、明天、一周后）
- ⏳ **倒计时**：选择日期，查看距离今天的天数
- 📈 **年度进度**：显示今年已过去的百分比

## 使用方法

### 时间戳转换
1. 在"日期时间"输入框中输入日期时间，自动转换为时间戳
2. 或在"时间戳"输入框中输入时间戳，自动转换为日期时间
3. 选择时区进行转换

### 多时区显示
- 自动显示多个常用时区的当前时间
- 实时更新，每秒刷新

### 快捷操作
- 点击"现在"：设置为当前时间
- 点击"今天"：设置为今天的 00:00:00
- 点击"明天"：设置为明天的 00:00:00
- 点击"一周后"：设置为 7 天后的时间

### 倒计时
- 选择目标日期（年、月、日）
- 自动显示距离今天的天数

## 技术实现

- 使用 [Luxon](https://moment.github.io/luxon/) 库处理日期时间
- 支持所有 IANA 时区
- 纯 HTML/CSS/JavaScript 实现
- 响应式设计，适配移动设备
